Election Sim
------------

This requires:
    * PyGTK
    * PyProcessing (the multiprocessing library in Python 2.6)

On Linux you can get these through your package manager.  Windows users can get 
these at the following locations:
    * Python: http://www.python.org/ftp/python/2.5.2/python-2.5.2.msi
    * GTK: http://superb-east.dl.sourceforge.net/sourceforge/gladewin32/gtk-2.12.9-win32-2.exe
    * PyCairo: http://ftp.gnome.org/pub/GNOME/binaries/win32/pycairo/1.4/pycairo-1.4.12-1.win32-py2.5.exe
    * PyGobject: http://ftp.gnome.org/pub/GNOME/binaries/win32/pygobject/2.14/pygobject-2.14.1-1.win32-py2.5.exe
    * PyGTK: http://ftp.gnome.org/pub/GNOME/binaries/win32/pygtk/2.12/pygtk-2.12.1-2.win32-py2.5.exe
    * PyProcessing: http://prdownload.berlios.de/pyprocessing/processing-0.52.win32-py2.5.exe
